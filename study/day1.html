html 구성

    html은 크게 head와 body로 구성되어있다.
    
    head는 meta태그 및, style, script 의 link들로 이루어져 있고, title을 포함하고있다.
    meta태그의 종류는 다양하며, 웹사이트의 조건에 맞게 설정해 줘야한다. charset은 꼭 들어가야한다.
    
    body 영역은 웹사이트를 구성할 태그들로 이루어져있다.

태그의 종류

태그는 크게 인라인태그 및 블록태그로 나뉘어 있다. 인라인 태그는 높이와 너비값을 가지고 있지 않는 특징이 있어, 인라인 태그 안의 text 값만큼 부피를 차지한다. 반면에 블록태그는 너비 100%로 크기를 가지고 있다.

  • 인라인 태그 종류

    • 링크 : a
    • 강조 : span, em, i, strong, b, del, s, u
    • 테이블 : table, thead, tbody, tfoot, tr, th, td
    • 이미지 : img
    • 첨자 : sub, sup
    • 폼 : input, button, label, select, option
    • 줄바꿈 : br

  • 블록 태그 종류

    • 그룹 : div, section, aside, header, mian, footer, article
    • 제목 : h1, h2, h3, h4, h5, h6
    • 내용 : p
    • 리스트 : ul, ol, li, dl, dt, dd
    • 폼 : form, fieldset, textarea, pre
